๐ฌ๐ง Travelling Salesman Problem (TSP) Definition
The Travelling Salesman Problem (TSP) is one of the most fundamental problems in combinatorial optimization and graph theory. Given a set of cities and the distances or travel costs between each pair of cities, the objective is to find a route that:
- Visits every city exactly once,
- Returns to the starting city,
- And has the minimum possible total cost or distance.
TSP is classified as an NP-hard problem, meaning that no known algorithm can compute the exact optimal solution in polynomial time for large instances. Therefore, various exact, approximate, and metaheuristic methods are used to solve it.
